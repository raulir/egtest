
Ülesanne on teha paneel mobiilseks.

Sinine kast on mobiiliekraani laius, sellest väljaspoole jääv ala on normaalses olukorras peidetud.

Oluline:

* Muuta ainult html failis muudetava ala sisse jäävat koodi, css ja js failides samuti ainult märgitud ala sisse jäävat osa. Eeldada, et alati on 3 pisipilti.

* Palun võimalikult täpset jäljendust - palun mitte lisada jooni jms elemente, kuna just täpselt selline disain on kliendi tähtsate isikute komitee poolt kinnitatud.

* Kõik muutujate nimed, klassinimed jne peavad olema prefikseeritud paneeli nimega. Antud juhul "featured".

* Muutujate nimed ainult väiketähtedega ja liigendatud ainult alakriipsudega. Sama ka klassinimede jms kohta.

* Teksti astendamiseks kasutada 4 tühikut või tabulatsiooni.

* Üritada järgida stiili ka muudes elementides, palun. See vähendab arendusel mitteolulist mõttetööd!

* Kasutada ainult rem ühikuid või protsente.

* Kasutada ainult div elemente

* Ekraani laius mobiilses vaates on alati 51.2rem!



Ülesanne:

1. Peale lehekülje laadimist asetsevad pealkirjad ja pisipildid vastavalt mobile.png asetusele. 

2. Lisada kolm nuppu vastavalt pildile.

3. Kui klikitakse nupul A, siis animeeritakse sisu paremale nagu animation.jpg 2. joonis. B ja C klikkimine siis animeerib vastavalt vasakule ja keskele tagasi. Animeerimiseks palun kasutada ".featured_container" elemendile lisatavaid/eemaldatavaid klasse ja css animatsiooni.
